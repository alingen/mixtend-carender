## Getting Started
```
git clone git@github.com:alingen/mixtend-carender.git
```
```
cd mixtend-carender
```
```
cp .env{.example,}
```
```
php artisan key:generate
```
```
php artisan migrate
```
```
npm install
```
```
npm run dev
```
```
composer install
```
```
php artisan serve
```
